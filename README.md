# ZapretManager

Windows-приложение для установки, обновления, запуска, проверки и отката сборки [Flowseal/zapret-discord-youtube](https://github.com/Flowseal/zapret-discord-youtube) на Windows.

> `ZapretManager` нужен для того, чтобы работать со сборкой zapret из одного окна: без постоянного переключения между bat-файлами, папками, списками доменов и ручной диагностикой.

## Что это

`ZapretManager` работает именно со сборкой [Flowseal/zapret-discord-youtube](https://github.com/Flowseal/zapret-discord-youtube), а не с произвольной папкой с любыми bat-файлами.

Это отдельный Windows-менеджер, который помогает:

- найти или скачать рабочую сборку zapret
- запустить и сравнить конфиги
- установить лучший вариант как службу
- поддерживать DNS, IPSet, списки, `TG WS Proxy` и диагностику в одном месте

## Что умеет программа

- автоматически или вручную находить рабочую папку zapret
- скачивать, обновлять, удалять и откатывать сборку
- запускать выбранный конфиг и останавливать `winws`
- устанавливать, удалять и автоматически переносить службу на новую версию программы
- подбирать конфиг через автоматический режим
- автоматически устанавливать `zapret` в managed-папку при первом автоматическом режиме, если сборка ещё не подключена
- проверять конфиги по HTTP, TLS 1.2, TLS 1.3 и ping
- показывать подробные результаты проверки по каждой цели
- скрывать нерабочие конфиги и возвращать их из отдельного окна
- редактировать домены для проверки, `hosts`, включённые и исключённые домены, `IPSet`
- управлять DNS-профилями, пользовательским DNS, DoH, режимом `IPSet` и игровым режимом
- устанавливать, обновлять, запускать, останавливать и удалять `TG WS Proxy`
- проверять и обновлять `TG WS Proxy` через GitHub releases, управлять его автозапуском, логом и `CF-proxy`
- запускать диагностику системы с пояснениями по проблемным местам
- очищать кэш Discord из интерфейса программы
- работать через трей, в single-instance режиме и с темой `Светлая / Тёмная / Как в системе`
- автоматически проверять обновления по настраиваемому интервалу
- автоматически обновлять список `IPSet` при запуске, если это включено в настройках

## Почему это удобно

- не нужно вручную вспоминать, какой bat-файл запускать
- можно быстро сравнить конфиги и увидеть, что реально работает лучше
- служба, DNS, IPSet, `TG WS Proxy`, диагностика и списки управляются из одной программы
- обновление и откат сборки делаются без ручной возни с папками
- после обновления и отката сборки не нужно вручную восстанавливать службу в типичных сценариях

## Если не хочется разбираться вручную

Если вообще не хочется вникать в конфиги, bat-файлы и дополнительные настройки, в программе есть `Автоматический режим (zapret)` и отдельный `Автоматический режим (TG WS Proxy)`.

Как это работает на практике:

- если сборка `zapret` ещё не установлена, программа сама скачает её в managed-папку `Components`
- если сборка уже подключена, этот шаг пропускается
- автоматически подготавливает окружение: обновляет `IPSet` и `hosts`
- запускает полную проверку доступных конфигов
- сравнивает результаты HTTP, TLS 1.2, TLS 1.3 и ping
- выбирает лучший полностью подходящий конфиг
- сразу устанавливает его как службу Windows и запускает её

Для `TG WS Proxy` автоматический режим:

- проверяет наличие или устанавливает `TG WS Proxy`
- при необходимости обновляет его до актуального релиза
- включает автозапуск
- подготавливает ссылку `tg://proxy`
- открывает Telegram с готовым прокси

Во многих случаях для первого старта достаточно просто запустить программу и нажать `Автоматический режим`.

## Для кого это

`ZapretManager` подойдёт тем, кто:

- использует zapret на Windows
- хочет быстро запускать и тестировать разные конфиги
- не хочет вручную редактировать служебные файлы в папке сборки
- хочет держать DNS, диагностику и списки целей под рукой

## Быстрый старт

1. Скачайте `ZapretManager.exe` из раздела [Releases](https://github.com/Valturere/zapretmanager-desktop/releases).
2. Запустите программу.
3. Укажите существующую папку `zapret` или скачайте свежую сборку через интерфейс.
4. Выберите конфиг и проверьте его, либо сразу запустите `Автоматический режим`.
5. При необходимости настройте `TG WS Proxy` во вкладке `Telegram / Proxy`.

## Что есть в интерфейсе

Основные сценарии собраны в одном месте:

- `Главная`:
  - общее состояние компонентов
  - быстрые режимы
  - карточки службы и `Telegram`
- `Конфиги`:
  - обычная проверка
  - режим `TCP 16-20`
  - список доменов для проверки
  - результаты и таблица конфигов
- `Установка и обновление`:
  - `zapret`
  - `TG WS Proxy`
  - `ZapretManager`
- `DNS и инструменты`:
  - настройка DNS-адреса
  - `IPSet`
  - игровой режим
  - обслуживание
  - списки и редакторы
- `Telegram / Proxy`:
  - `MTProto`
  - `TG WS Proxy`
  - датацентры Telegram
  - `Cloudflare Proxy`
- `Настройки`:
  - поведение программы
  - тема приложения
  - настройка обновлений
  - параметры `TG WS Proxy`

## Работа через трей

Программа умеет нормально жить в системном трее:

- сворачиваться в трей вместо полного закрытия
- быстро открываться обратно по значку
- давать быстрый доступ к основным действиям
- не мешать в панели задач, если хочется оставить её работать в фоне
- запускаться скрыто через планировщик задач при автозапуске

Это удобно, если zapret уже запущен и программу нужно просто держать под рукой.

## Системные требования

- Windows 10 / Windows 11
- x64
- права администратора для части действий со службой, драйвером и системными файлами

## Важно про Защитник Windows

Сам `ZapretManager` обычно не вызывает предупреждений антивируса, но Защитник Windows может ругаться на файлы самой сборки zapret.

Если во время установки, обновления или запуска сборки что-то блокируется, лучше:

- добавить папку сборки zapret в исключения Защитника Windows
- либо временно отключить защиту на время установки или обновления сборки через программу

Это относится именно к файлам zapret, а не к интерфейсу `ZapretManager`.
