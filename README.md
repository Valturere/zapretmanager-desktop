# ZapretManager

Desktop-приложение для Windows, которое упрощает установку, обновление, запуск и проверку сборок zapret на базе [Flowseal/zapret-discord-youtube](https://github.com/Flowseal/zapret-discord-youtube).

## Что это такое

`ZapretManager` помогает работать со сборкой zapret без ручного поиска нужных bat-файлов, редактирования списков и постоянного переключения между папками. Программа берёт на себя рутинные действия: поиск сборки, обновление, запуск конфигов, диагностику, DNS-настройки и проверку результатов.

## Основные возможности

- автоматический и ручной поиск папки zapret
- скачивание, обновление, удаление и откат сборки
- запуск выбранного конфига и остановка `winws`
- установка и удаление службы Windows
- автоматический режим с подбором подходящего конфига
- проверка конфигов по HTTP, TLS 1.2, TLS 1.3 и ping
- подробные результаты проверки по каждой цели
- скрытие нерабочих конфигов и отдельное окно скрытых конфигов
- редактирование `targets.txt`, включённых и исключённых доменов, `hosts`, пользовательских подсетей IPSet
- встроенные DNS-профили, пользовательский DNS, DoH и DNS-диагностика
- светлая и тёмная тема
- работа через трей и single-instance режим

## Для кого программа

Программа подойдёт тем, кто:

- использует сборки zapret на Windows
- не хочет каждый раз вручную запускать разные конфиги
- хочет быстро проверить, какой конфиг реально работает лучше
- хочет управлять DNS, списками и службой из одного окна

## Системные требования

- Windows 10 / Windows 11
- x64
- права администратора для части операций со службой, драйвером и системными файлами

## Как начать

1. Скачай `ZapretManager.exe` из раздела **Releases**.
2. Запусти программу.
3. Укажи существующую папку zapret или скачай свежую сборку через интерфейс.
4. Выбери конфиг и проверь его, либо запусти автоматический режим.

## Статус проекта

Первая публичная версия готовится как `1.0`. Для внутренних рабочих сборок используется отдельная схема вида `1.0-dev01`, `1.0-dev02` и так далее.

## Важно

`ZapretManager` не заменяет сам проект zapret и не является его официальной частью. Это отдельный менеджер для удобной работы со сборками на Windows.
